# Magic Item Tools
*Source: Dungeon Master's Guide (2024) p. 220* 

**Magic Item Tools**

| Item Category | Required Tool |
|---------------|---------------|
| Armor | [Leatherworker's Tools](/items/leatherworkers-tools-xphb.md), [Smith's Tools](/items/smiths-tools-xphb.md), or [Weaver's Tools](/items/weavers-tools-xphb.md) depending on the kind of armor as noted in the tools' descriptions |
| Potion | [Alchemist's Supplies](/items/alchemists-supplies-xphb.md) or [Herbalism Kit](/items/herbalism-kit-xphb.md) |
| Ring | [Jeweler's Tools](/items/jewelers-tools-xphb.md) |
| Rod | [Woodcarver's Tools](/items/woodcarvers-tools-xphb.md) |
| Scroll | [Calligrapher's Supplies](/items/calligraphers-supplies-xphb.md) |
| Staff | [Woodcarver's Tools](/items/woodcarvers-tools-xphb.md) |
| Wand | [Woodcarver's Tools](/items/woodcarvers-tools-xphb.md) |
| Weapon | [Leatherworker's Tools](/items/leatherworkers-tools-xphb.md), [Smith's Tools](/items/smiths-tools-xphb.md), or [Woodcarver's Tools](/items/woodcarvers-tools-xphb.md) depending on the kind of weapon as noted in the tools' descriptions |
| Wondrous Item | [Tinker's Tools](/items/tinkers-tools-xphb.md) or the tool required to make the nonmagical item on which the magic item is based |
