# Orb of Direction
*Wondrous item, common*  
![](https://raw.githubusercontent.com/5etools-mirror-3/5etools-img/main/items/XDMG/Orb%20of%20Direction.webp#right)

- **Weight**: 3.0 lbs.

This orb can be used as an Arcane Focus.

While holding this orb, you can take a [Magic](/rules/actions.md#Magic) action to determine which way is magnetic north. Nothing happens if the orb is used in a location that has no magnetic north.

*Source: Dungeon Master's Guide (2024) p. 283*
