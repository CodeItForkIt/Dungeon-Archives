# Scroll of Protection
*Scroll, rare*  
![](https://raw.githubusercontent.com/5etools-mirror-3/5etools-img/main/items/XDMG/Scroll%20of%20Protection.webp#right)


Each Scroll of Protection works against creatures of a specific creature type chosen by the DM or determined by rolling on the following table.

| dice: 1d100 | Creature Type |
|-------------|---------------|
| 01-10 | [Aberrations](/items/scroll-of-protection-aberrations-xdmg.md) |
| 11-15 | [Beasts](/items/scroll-of-protection-beasts-xdmg.md) |
| 16-20 | [Celestials](/items/scroll-of-protection-celestials-xdmg.md) |
| 21-25 | [Constructs](/items/scroll-of-protection-constructs-xdmg.md) |
| 26-35 | [Dragons](/items/scroll-of-protection-dragons-xdmg.md) |
| 36-45 | [Elementals](/items/scroll-of-protection-elementals-xdmg.md) |
| 46-50 | [Humanoids](/items/scroll-of-protection-humanoids-xdmg.md) |
| 51-60 | [Fey](/items/scroll-of-protection-fey-xdmg.md) |
| 61-70 | [Fiends](/items/scroll-of-protection-fiends-xdmg.md) |

| 76-80 | [Monstrosities](/items/scroll-of-protection-monstrosities-xdmg.md) |
| 81-85 | [Oozes](/items/scroll-of-protection-oozes-xdmg.md) |
| 86-90 | [Plants](/items/scroll-of-protection-plants-xdmg.md) |
| 91-00 | [Undead](/items/scroll-of-protection-undead-xdmg.md) |
^1-creature-type

Using a [Magic](/rules/actions.md#Magic) action to read the scroll creates a 5-foot Emanation originating from you. For 5 minutes, creatures of the specified type can't enter or affect anything in the area. However, if you move in such a way that a creature of the specified type would be inside the area, the effect ends.

As a [Magic](/rules/actions.md#Magic) action, a creature within 5 feet of the Emanation can attempt to overcome it, which forces the creature to make a DC 15 Charisma saving throw. On a successful save, the creature ceases to be affected by the Emanation.
